# Color-Palettes
The base project is to create color combinations based on randomly generated numbers and color theory for agreeable colors alongside.

# Features
- A Colorwheel to select a color
- Random Color Generator
- Palette-creation based on color theory
- Add and Remove cards (choice of removal)
- Drag-drop rearranging

# Tools
This project is built using Javascript, HTML, and CSS. This is completely plain code.
